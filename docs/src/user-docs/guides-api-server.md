# API Server

The API Server exposes a simple, REST-style interface on top of the CLP data-backend. It lets you:

* Submit search queries against CLP data.

* Stream query results via `text/event-stream` format.

The API is defined using the OpenAPI Specification. You can check the API via [Swagger UI][swagger].

[swagger]: https://petstore.swagger.io/?url=https://docs.yscope.com/clp/main/_static/openapi.json
